Introducción a HTML

Definición

HTML son las siglas de Hyper Text Markup Language, que significa lenguaje de marcas (o de marcado o de etiquetas) de hipertextos (hiperenlaces o hipervínculos). El lenguaje de los ficheros de texto plano HTML es leído por el navegador (que hace el renderizado) y así se pueden percibir las páginas web. Otros ejemplos de lenguaje de marcado es xml.

Propiedades

  • Es un lenguaje declarativo porque declara lo que se quiere hacer mediante etiquetas. Ej.: La etiqueta

    sirve para hacer un párrafo.
  • No es un lenguaje de programación porque no permite crear algoritmos ni variables.
  • Tiene una sintaxis propia.
  • Es un lenguaje sin estado, por lo que no guarda información, no tiene cookies (que aparecen con javascript).
  • De los pocos lenguajes insensibles al uso de minúsculas o mayúsculas, pero no se recomienda el uso generalizado de mayúsculas.

La sintaxis básica de HTML es:

                
                    <nombreetiqueta> Contenido </nombreetiqueta>
